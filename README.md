## Arch Hyprland -- *The Switch From Windows*

Extremely personalized configs. Use at your own risk.

## USAGE

1. `cd ~`
2. `git clone https://github.com/SMCaseyCode/dotfiles.git`
3. `cd dotfiles/scripts`
4. `chmod +x initial-setup.sh`
5. `./initial-setup.sh`

**WARNING**: Some applications require absolute paths so some theming may need to be tinkered with
unless if your user is named the same as mine.

## INCLUDED DOTFILES

- `fastfetch:` customizations for the output of `fastfetch`
- `hypr:` configuration for hyprland dynamic tiling compositor; Lua and hyprlang included
- `kitty:` configuration for terminal
- `waybar:` configuration for waybar (status bar)
- `wireplumber:` rename config for Rodecaster Duo Device
- `matugen:` Configs & Templates for generating color schemes based on wallpaper
- `nvim:` Configs for neovim & plugins
- `swaync:` Configs for notifcation handler
- `wofi:` Configs for application menu (wayland version of Rofi)
- `yazi:` Configs for yazi; terminal filebrowser
